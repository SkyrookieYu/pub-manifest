Manifest to Internal Representation Processor

This app shows the result of converting a Publication Manifest into the expected internal representation. The processor is written in JavaScript and is structured to match the processing algorithm defined in the Publication Manifest specification.

This page needs to be hosted from a web server in order to work. To test other manifests, modify the manifest.json file in this directory and refresh this page. Use the developer console in your browser to view any errors and warnings that result from processing.
